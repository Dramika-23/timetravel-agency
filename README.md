# TimeTravel Agency — Webapp Interactive

Webapp pour une agence de voyage temporel fictive, créée avec IA générative.
Elle met en scène l'agence et ses 3 destinations (Paris 1889, Crétacé -65M,
Florence 1504) avec une interface immersive et un agent conversationnel IA.

## 🛠️ Stack technique
- React + Tailwind CSS
- API Mistral (chatbot)
- Hébergement : Vercel

## ✨ Features
- Landing page interactive (hero, présentation de l'agence)
- Galerie de 3 destinations temporelles (cards interactives)
- Chatbot IA conversationnel ("Chronos") qui conseille les visiteurs
- Design responsive mobile-first, dark mode premium

## 🤖 IA utilisées
- Code : Bolt.new (vibe coding)
- Chatbot : Mistral Small via API
- Visuels : illustrations générées (set Paris / Crétacé / Florence)

## ⚙️ Installation
1. Cloner le repo
2. `npm install`
3. Créer un fichier `.env` avec votre clé : `MISTRAL_API_KEY=votre_clé`
4. `npm run dev`
